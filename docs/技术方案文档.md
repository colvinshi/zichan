# 资产管理系统技术方案

## 1. 系统概述

资产管理系统是一套企业级资产管理解决方案，支持资产的全程管理、盘点任务执行移动端扫码盘点等功能。系统采用微服务架构设计，分为后端服务、管理后台、移动端三大模块。

### 1.1 系统架构

```

│                        用户层                                │

│  │  Vue3+Element  │  │   uni-app      │  │   uni-app   │  │
```

### 1.2 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| 用户管理 | 用户CRUD、角色管理、密码重置、部门管理 |
| 资产管理 | 资产增删改查、资产分类、资产转移、资产统计 |
| 盘点管理 | 盘点任务创建、任务执行、结果提交、差异报告 |
| 移动端盘点 | 扫码识别资产、实时盘点、离线支持 |

## 5. 安全设计

### 5.1 认证授权

- **认证方式**: JWT无状态令牌
- **访问令牌**: 有效期2小时
- **刷新令牌**: 有效期7天
- **签名算法**: HS512
- **密码加密**: BCrypt

### 5.2 角色权限

| 角色编码 | 角色名称 | 权限 |
|---------|---------|------|
| admin | 管理员 | 全部权限 |
| asset_manager | 资产管理员 | 资产管理、盘点管理 |
| employee | 普通员工 | 查看个人资产、执行盘点 |

### 5.3 安全防护

- 接口限流(普通用户100次/分钟，管理员200次/分钟)
- 敏感数据加密存储(AES)
- 数据脱敏展示(手机号、身份证号)
- SQL注入防护(参数化查询)
- XSS防护(输入过滤)

## 9. 附录

### 9.1 角色说明

- **管理员(admin)**: 系统最高权限，可管理所有用户和资产
- **资产管理员(asset_manager)**: 负责资产管理和盘点任务管理
- **普通员工(employee)**: 查看个人资产，执行盘点任务

### 9.2 资产状态说明

| 状态码 | 状态名 | 说明 |
|--------|--------|------|
| 1 | 使用中 | 资产已被领用 |
| 2 | 闲置 | 资产未分配 |
| 3 | 维修中 | 资产正在维修 |
| 4 | 已报废 | 资产已报废 |

### 9.3 盘点结果说明

| 结果码 | 结果名 | 说明 |
|--------|--------|------|
| 1 | 一致 | 账面与实际相符 |
| 2 | 盘盈 | 实际多了资产 |
| 3 | 盘亏 | 实际少了资产 |
| 4 | 损坏 | 资产损坏 |
