# 资产管理系统 - 使用手册

## 1. 系统简介

资产管理系统是一套企业级资产管理解决方案，提供资产全生命周期管理、盘点任务执行移动端扫码等功能。

### 1.1 系统功能模块

| 模块 | 功能 |
|------|------|
| 首页 | 资产统计、待办任务展示 |
| 资产管理 | 资产增删改查、分类管理、转移记录 |
| 盘点管理 | 盘点任务创建、执行、结果审核 |
| 用户管理 | 用户CRUD、角色分配(管理员) |
| 我的资产 | 个人名下资产查看 |
| 移动端 | 扫码盘点、资产查询 |

---

## 2. 快速开始

### 2.1 启动后端服务

```bash
# 1. 进入后端目录
cd asset-parent

# 2. 配置数据库
# 创建数据库: CREATE DATABASE asset_db;
# 修改 application.yml 中的数据库连接信息

# 3. 启动服务
mvn spring-boot:run
```

服务启动后访问: http://localhost:8080

### 2.2 启动管理后台

```bash
# 1. 进入前端目录
cd asset-web

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

管理后台启动后访问: http://localhost:3000

### 2.3 启动移动端(H5)

```bash
# 1. 进入移动端目录
cd asset-mobile

# 2. 安装依赖
npm install

# 3. 启动H5
npm run dev:h5
```

移动端启动后访问: http://localhost:5173

## 4. 功能操作指南

### 4.1 资产管理

#### 4.1.1 查看资产列表

1. 点击左侧菜单「资产管理」
2. 可通过关键词、状态进行筛选
3. 支持分页查看

#### 4.1.2 创建资产

1. 点击「新增资产」按钮
2. 填写资产信息:
   - 资产编码 (必填，唯一)
   - 资产名称 (必填)
   - 资产分类 (必填)
   - 品牌型号
   - 序列号
   - 采购信息
3. 点击「确定」保存

#### 4.1.3 编辑资产

1. 在资产列表点击「编辑」按钮
2. 修改资产信息
3. 点击「确定」保存

#### 4.1.4 删除资产

1. 在资产列表点击「删除」按钮
2. 确认删除操作
3. 资产将被逻辑删除

#### 4.1.5 资产调拨

1. 点击资产「调拨」按钮
2. 选择目标使用人和部门
3. 填写备注信息
4. 确认调拨

### 4.2 盘点管理

#### 4.2.1 创建盘点任务

1. 点击「盘点任务」
2. 点击「创建任务」
3. 填写任务信息:
   - 任务名称
   - 开始/结束时间
   - 盘点范围(部门/人员)
   - 任务类型(全面/部分)
4. 点击「确定」

#### 4.2.2 执行盘点

**管理后台执行:**
1. 找到进行中的任务
2. 点击「执行」
3. 扫描或输入资产编码
4. 选择盘点结果
5. 提交盘点

**移动端执行:**
1. 打开移动端「盘点」页面
2. 点击扫码图标扫描资产二维码
3. 选择盘点结果(一致/盘盈/盘亏/损坏)
4. 点击「提交盘点结果」

#### 4.2.3 完成任务

1. 盘点完成后，点击「完成」
2. 系统生成盘点差异报告
3. 资产状态根据盘点结果更新

### 4.3 用户管理(管理员)

#### 4.3.1 创建用户

1. 点击「用户管理」
2. 点击「新增用户」
3. 填写用户信息:
   - 用户名 (必填，唯一)
   - 密码
   - 真实姓名
   - 部门
   - 角色
4. 点击「确定」

#### 4.3.2 管理用户状态

- 点击「禁用」可冻结用户账号
- 点击「启用」可恢复正常
- 点击「重置密码」可重置为默认密码

### 4.4 我的资产

1. 点击「我的资产」
2. 查看个人名下资产列表
3. 可查看资产详情
4. 可进行资产归还操作

## 5. 移动端操作指南

### 5.1 登录

1. 打开移动端应用

3. 点击「登录」

- 查看资产统计卡片

1. 点击「扫码盘点」
2. 扫描资产二维码 (微信扫码)
3. 或手动输入资产编码
4. 选择盘点结果
5. 提交盘点

### 5.4 我的资产

1. 点击「资产」
2. 查看个人资产列表
3. 点击查看详情

## 6. 常见问题

### 6.1 登录问题

**Q: 登录提示"用户名或密码错误"**
A: 检查账号密码是否正确，默认账号: admin / Admin@123

**Q: 登录提示"账号已禁用"**
A:请联系管理员启用账号

### 6.2 资产问题

**Q: 创建资产提示"编码已存在"**
A: 资产编码必须唯一，请使用其他编码

**Q: 无法删除资产**
A: 检查是否有盘点任务关联该资产

### 6.3 盘点问题

**Q: 盘点任务无法开始**
A: 任务状态必须为"待开始"才能开始

**Q: 扫码无反应**
A: 请在微信小程序中使用扫码功能
